# Sprint 5 Plan – Rendering Layer

## Mục tiêu
Sinh ra bản preview infographic đầu tiên từ specification đã được chuẩn hóa.

## Deliverable chính
- Rendering service đầu tiên.
- Lưu ảnh/asset preview vào storage.
- API và UI hiển thị preview.
- Trạng thái render và lỗi render cơ bản.

## Công việc thực hiện
- Backend/Render: hiện thực pipeline render từ spec.
- Frontend: màn hình preview và loading/error state.
- QA: test hiển thị, fallback khi asset thiếu hoặc render fail.

## Rủi ro
- Chất lượng render ban đầu chưa ổn.
- Asset đầu ra nặng hoặc thời gian tạo lâu.
